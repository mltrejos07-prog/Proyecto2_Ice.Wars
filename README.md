# Proyecto2_Ice.Wars
# Descripción
Juego de estrategia por turnos para dos jugadores desarrollado en Python.
Un jugador asume el rol de defensor construyendo torres y muros, mientras
el otro asume el rol de atacante intentando destruir la base central.
El primero en ganar 3 rondas gana la partida.

# Requisitos
-Python 3.9
-Pillow: `pip install pillow`
-Pygame: `pip install pygame`

# Cómo ejecutar
1. Cloná o descargá el repositorio
2. Instalá las dependencias: "pip install pillow pygame"
3. Ejecutá el archivo principal: "python "Main code.py"

# Estructura del proyecto
-"Main code.py": archivo principal
-"obj/":carpeta con imágenes y música
-"Jugadores.json": se crea automáticamente al registrar jugadores

# Controles
-Clic en el mapa para colocar unidades o defensas
-Panel izquierdo: defensor
-Panel derecho: atacante
